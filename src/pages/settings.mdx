## Get Settings {{ tag: 'GET', label: '/v1/settings' }}

<Row>
  <Col>

    This endpoint allows you to retrieve the application settings. The response includes details such as business information, policies, contact details, and other configurations.

    ### Edge Cases:
    - **No Settings Found**: If no settings exist, the server will return an empty object with a `200 OK` status code.

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="api/v1/settings">

    ```bash {{ title: 'HTTP(S)' }}
    https:///{{host}}/api/v1/settings
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "data": {
        "name": "APP_SETTINGS",
        "businessName": "My Business",
        "businessCategory": "E-commerce",
        "businessDescription": "An online store selling products.",
        "address": "123 Business St.",
        "addressDescription": "Main office location.",
        "locationName": "Headquarters",
        "email": "contact@mybusiness.com",
        "phone": "+1234567890",
        "showTermsAndConditions": true,
        "showPrivacyPolicy": true,
        "showReturnPolicy": true,
        "showDigitalItemPolicy": true,
        "showContactUs": true,
        "sendOrderConfirmationEmailAutomatically": true,
    ```

## Reset Settings {{ tag: 'DELETE', label: '/api/v1/settings' }}

<Row>
  <Col>

    This endpoint allows you to reset the application settings to their default values. It will revert any changes made to the settings back to the original configuration.

    ### Edge Cases:
    - **No Settings to Reset**: If no settings exist, the server will return a `404 Not Found` status code.

  </Col>
  <Col sticky>
    <CompactTable heading="Headers" hasHeader={false}>
      <CompactRow>
        <CompactCell>Authorization</CompactCell>
        <CompactCell isLast>Bearer `{{admin_token}}` </CompactCell>
      </CompactRow>
    </CompactTable>

    <CodeGroup title="Request" tag="DELETE" label="/api/v1/settings">

    ```bash {{ title: 'HTTP(S)' }}
    https://{{host}}/api/v1/settings
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "status": 200,
      "message": "Settings have been reset to default values."
    }
    ```

  </Col>
</Row>
